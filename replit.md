# Prompt to iOS App Generator

## Overview

A full-stack developer tool that converts natural language prompts into production-ready iOS app code. Users describe their app idea, choose SwiftUI or UIKit, and get a complete multi-file Xcode-ready project generated by AI.

## Stack

- **Monorepo tool**: pnpm workspaces
- **Node.js version**: 24
- **Package manager**: pnpm
- **TypeScript version**: 5.9
- **Frontend**: React + Vite (artifacts/promptios) — dark IDE aesthetic
- **API framework**: Express 5 (artifacts/api-server)
- **Database**: PostgreSQL + Drizzle ORM
- **Validation**: Zod (`zod/v4`), `drizzle-zod`
- **API codegen**: Orval (from OpenAPI spec)
- **AI**: OpenAI via Replit AI Integrations (gpt-5.3-codex for code generation)
- **Build**: esbuild (CJS bundle)

## Features

- **Dashboard**: Stats (total projects, files generated, framework breakdown), recent projects
- **New Project**: Form to enter app name, prompt description, and framework (SwiftUI/UIKit)
- **Project Detail**: File tree + syntax-highlighted code viewer, SSE streaming generation
- **AI Generation**: Streams SwiftUI/UIKit code via SSE, parses multi-file JSON response

## Key Commands

- `pnpm run typecheck` — full typecheck across all packages
- `pnpm run build` — typecheck + build all packages
- `pnpm --filter @workspace/api-spec run codegen` — regenerate API hooks and Zod schemas from OpenAPI spec
- `pnpm --filter @workspace/db run push` — push DB schema changes (dev only)

## Database Tables

- `projects` — project metadata (name, prompt, status, framework, file_count)
- `project_files` — generated Swift files (filename, filepath, content, language)

## API Endpoints

- `GET /api/projects` — list all projects
- `POST /api/projects` — create project
- `GET /api/projects/recent` — 5 most recent
- `GET /api/projects/stats` — aggregate stats
- `GET /api/projects/:id` — get project
- `DELETE /api/projects/:id` — delete project
- `GET /api/projects/:id/files` — get generated files
- `POST /api/projects/:id/generate` — SSE stream code generation

## Environment Variables

- `DATABASE_URL` — PostgreSQL connection string
- `AI_INTEGRATIONS_OPENAI_BASE_URL` — Replit AI proxy URL
- `AI_INTEGRATIONS_OPENAI_API_KEY` — Replit AI proxy key
- `SESSION_SECRET` — session secret

See the `pnpm-workspace` skill for workspace structure, TypeScript setup, and package details.
